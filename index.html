FAQs

What do you mean by accessible programming?

Programming is usually intimidating to a lot of beginners and non-technical folks. My goal is to make programming accessible - easy to approach and fun to learn.

And, yes, it can be done. Even if you've never programmed before. Even if you feel you are not that good with maths or science.

Just like everybody can learn to read or write; everybody can learn to code (and get better with practice).

I am an entrepreneur. Should I learn how to code?

If you are a solopreneuer/ entrepreneur/ business-owner, then you should definitely learn how to code. At least build and control your own business website (start with workshop/STATICK).

If you want to build your own tech-startup or app, then you obviously need to learn how to code. Even if it's just enough to launch the initial MVP/ prototype (launching soon workshop/METEORIK).

If you are an entrepreneur/ founder/ product manager who works with technical teams or software freelancers, you should learn the basics of software development. This will help you communicate better with technical team (check out workshop/SOFTWEAR).

In any case, if you're even a little bit curious about programming, you should definitely drop in for my FreeClass sessions. These sessions are conducted every weekend, 3hrs/day, both online/offline.

What's a FreeClass and who should enroll?

FreeClass sessions are free weekend coding sessions (3hrs each sat/sun) for non-technical beginners, creatives and entrepreneurs who want to get started with programming.

These sessions are designed to introduce you to programming, sofware development and web development with minimal friction.

You should enroll if you have never coded or programmed before and have been wanting to start but didn't know how.

Each weekend session-pair (saturday/sunday) is stand-alone and self-sufficient but also unique. You can attend multiple weekends if needed.

What exactly will I learn in a FreeClass?

You will begin your coding journey using basic html, css, javascript and number of other beginner-friendly programming-languages/ tools like Scratch, Snap!, Bubble, Webflow etc.

Future sessions may also include playing around with tools that allow one to make art (music, stories, visuals etc) by writing code like processing.js/p5.js etc.

Everything we do in these sessions will be with the goal of giving you and intuitive understanding of coding concepts like variables, control-flow, databases, algorithms, data-structures etc.

Once you enroll/ pay for a workshop you'll receive venue details (offline slots) or hangout-invite (online slots).

After the workshop is over you get to retain the c9 student account and can reach out to me on gitter-chat/ facebook-group for post-workshop support.
