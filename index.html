FAQs Answered

What do you mean by accessible programming?

Programming is usually intimidating to a lot of beginners and non-technical folks. My goal is to make programming accessible - easy to approach and fun to learn.

And, yes, it can be done. Even if you've never programmed before. Even if you feel you are not that good with maths or science.

Just like everybody can learn to read or write; everybody can learn to code (and get better with practice).

I am an entrepreneur. Should I learn how to code?

If you are a single-person business owner/ entrepreneur/ founder, then you should definitely learn how to code. At least build and control your own business website (start with workshop/STATICK).

If you want to build your own tech-startup or app, then you obviously need to learn how to code. Even if it's just enough to launch the initial MVP/ prototype (launching soon workshop/METEORIK).

Even if you work with a technical co-founder or software freelancers, you should learn the basics of software development. This will help you communicate better with technical team (check out workshop/SOFTWEAR).

In any case, if you're even a little bit curious about programming, you should definitely drop in for my FreeClass sessions. These sessions are conducted every weekend, 3hrs/day, both online/offline.

What's a FreeClass and who should enroll?

FreeClass sessions are free weekend coding sessions (3hrs each sat/sun) for non-technical beginners, creatives and entrepreneurs who want to get started with programming.

These sessions are designed to introduce you to programming, sofware development and web development with minimal friction.

You should enroll if you have never coded or programmed before and have been wanting to start but didn't know how.

Each weekend session-pair (saturday/sunday) is stand-alone and self-sufficient but also unique. You can attend multiple weekends if needed.

What exactly will I learn in a FreeClass?

You will begin your coding journey using basic html, css, javascript and number of other beginner-friendly programming-languages/ tools like Scratch, Snap!, Bubble, Webflow etc.

Future sessions may also include playing around with tools that allow one to make art (music, stories, visuals etc) by writing code like processing.js/p5.js etc.

Everything we do in these sessions will be with the goal of giving you and intuitive understanding of coding concepts like variables, control-flow, databases, algorithms, data-structures etc.

What workshops are available, and which one should I enroll in ?

NOTE - At least one FreeClass attendance mandatory to enroll in any workshop.


1. Workshop/ STATICK

FOR - Those who want to build their own business (portfolio, sell events, sell products)

GOAL - Build and deploy your own business website to sell your own events ticketed-events & digital/physical goods (using satenndrra.com as a website template)

TECH - html, css, javascript, jquery, bootstrap, aws-s3, 3rd party services (google-forms, eventshigh, instamojo)

SKILLS - How to build websites using bootstrap templates, How to use javascript/jquery pugins, How to use 3rd party services for building fully functional websites


02. Workshop/ SOFTWEAR

FOR - Those who want to learn just enough software development & project management to be able to work better with software teams and freelancers

GOAL - Run a project execution simulation where participants act as business clients providing requirements/test criteria, reviewing effort estimation, validating deliverables & managing the whole project (using satenndrra.com website as a project case study)

TECH - lean startup principles, lean software development

SKILLS - Software development & project management fundamentals, Specifying requirements/ test-criteria, Estimating software effort, Validating software deliverables


03. Workshop/ METEORIK

FOR - Those who want to build and launch their own startup MVPs and apps

GOAL - Build and deploy a web/mobile/desktop app that allows your customers to register, login and access their dashboards (based on the satenndra.com app)

TECH - meteor, blaze, bootstrap, cordova, electron, rest API

SKILLS - How to build javascript client/server apps, How to create web/mobile/desktop apps from a single code base

How is the workshop conducted?

Once you enroll/ pay for a workshop you'll receive venue details (offline slots) or hangout-invite (online slots).

Before starting the workshop I will give you a c9 student account. You'll also need to join the gitter-chat room and the facebook-group.

The 3-hrs sessions will be very hands-on. At the end of the session you will have some code-assignments to do before you come for the next session (keep aside 2-3 hours/day for this).

After the workshop is over you get to retain the c9 student account and can reach out to me on gitter-chat/ facebook-group for post-workshop support.

What are On-Site Workshops and who can request these?

On-Site workshops are customized workshops that can be requested to be held at a specific location.

You can request these workshops if you are an organization (entrepreneur/ startup club, associations, incubators, accelerators etc) or an academic body (MBA colleges, engineering colleges, universities etc).

Use the Request / Workshop button to send me your requirements and I will get back to you with a customized workshop plan.

I conduct these workshops both in Bangalore as well as other cities (this subject to additional expenses of travel, food, lodging).

How can I customize an Onsite-Workshop for my organization?

You can customize the On-Site workshops for scope, duration & batch strength.

Scope customizations involve adding/ removing topics to/from a standard workshop module, changing the intensity of the course (like very in-depth etc) and combining separate modules to create a full coding bootcamp (yes, I do those as well).

Duration customization is linked to scope so this has to modified w.r.t the scope. Dates can be agreed upon based on mutual convenience.

Batch Strength customizations are straight forward. I need a minimum of 10 members/students to conduct an On-Site session. The maximum I can train (without impacting workshop quality) is 30 in a batch.
